# Video Scripts for Social Media

Four short scripts to launch with. Each is built for **vertical 9:16**
(TikTok / Reels / Shorts) and reusable as a horizontal cut for YouTube
Shorts or Twitter.

**Recording stack:** any screen recorder + your phone voice memo for VO
(I'd use QuickTime + Voice Memos, then assemble in CapCut or Descript).
Aim for natural cadence, not radio-host energy. The brand is calm,
considered — the videos should match.

**Common framing:**
- 1080×1920 vertical (or record at 1080×1920 in browser; Chrome
  DevTools' device-mode lets you set this exactly).
- Cursor visible (CleanShot / built-in macOS Pointer Mode).
- Background music: kept very low (-25 dB), instrumental, sparse.
  Suggestion: a quiet piano loop from Epidemic Sound's "Hopeful Acoustic"
  category. The app's own playback is the audio star — music should
  duck under it during chord previews.

## Production checklist

For each video, before posting:

- [ ] Audio levelled (VO at -6dB peak; music at -25dB; app audio at -12dB)
- [ ] Captions burned in (TikTok / Reels viewers default to sound-off)
- [ ] No personal data on screen (clear any saved songs not used in the
      demo)
- [ ] Cursor visible and clear
- [ ] Final 1-2 seconds: a clear "where to go" card (URL)
- [ ] Vertical 1080×1920; safe area (centered content, 200px top + bottom
      reserved for caption + UI chrome on each platform)
- [ ] Watermark / logo in a consistent corner (top-left)
- [ ] Filename pattern: `hit-me-<script-id>-<draft#>.mp4`
